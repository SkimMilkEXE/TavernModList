# TavernModList

A Terraria mod pack built from suggestions thrown around by me and my friends. Just a personal project made for fun, mostly it's a home for random ideas we come up with while playing and want to see in the game.

## What is this?

No grand theme, no roadmap, just whatever content sounds fun to add next.

## Building

This is a standard [tModLoader](https://github.com/tModLoader/tModLoader) mod. Open it from tModLoader's Mod Sources menu (or via the `.csproj`) and build like any other mod project.
